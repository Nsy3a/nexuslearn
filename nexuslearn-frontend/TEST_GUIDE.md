# Web3.0 AI学习平台 - 测试启动指南

## 🚀 项目概述

这是一个集成了Web3.0、AI和现代化设计的全栈学习平台，包含以下核心功能：

### ✨ 核心功能模块

1. **知识仓库系统**
   - 学习资料管理
   - 代码模块库
   - 上下文引擎
   - 智能搜索引擎

2. **AI社区系统**
   - 朋友圈互动
   - 兴趣圈交流
   - 话题圈讨论
   - AI助手集成

3. **学习管理中心**
   - AI驱动的个性化学习路径
   - 自适应评估系统
   - 智能练习系统
   - 学习进度可视化

4. **Web3.0集成**
   - MetaMask钱包连接
   - NFT学习证书
   - 智能合约交互
   - 去中心化身份验证

## 🔧 技术栈

- **前端**: Vue 3 + TypeScript + Vite
- **UI框架**: Element Plus + 自定义设计系统
- **状态管理**: Zustand
- **区块链**: Web3.js + MetaMask API
- **样式**: Tailwind CSS + 自定义CSS
- **图标**: Lucide React

## 🎯 快速启动

### 环境要求
- Node.js >= 16.0.0
- npm 或 pnpm
- MetaMask浏览器插件（用于Web3.0功能）

### 安装依赖
```bash
# 使用 pnpm（推荐）
pnpm install

# 或使用 npm
npm install
```

### 开发环境启动
```bash
# 启动开发服务器
pnpm dev

# 或使用 npm
npm run dev
```

### 生产环境构建
```bash
# 构建项目
pnpm build

# 预览构建结果
pnpm preview
```

## 📱 功能测试指南

### 1. 钱包连接测试
1. 确保已安装MetaMask插件
2. 点击右上角"连接钱包"按钮
3. 选择MetaMask或其他钱包
4. 授权连接请求

### 2. 学习路径测试
1. 进入"学习管理"页面
2. 设置学习目标（如：通过CPA考试）
3. 添加核心技能标签
4. 点击"AI生成计划"按钮
5. 查看生成的学习路径

### 3. 智能练习测试
1. 在"学习管理"页面配置练习参数
2. 选择练习主题和难度
3. 点击"开始练习"按钮
4. 完成题目并提交答案
5. 查看AI反馈和建议

### 4. 社区功能测试
1. 进入"AI社区"页面
2. 发布学习动态或问题
3. 参与话题讨论
4. 使用AI助手获取建议
5. 加入兴趣圈子

### 5. 知识仓库测试
1. 访问"知识仓库"相关页面
2. 搜索学习资料或代码模块
3. 使用高级搜索功能
4. 上传和管理资源
5. 利用上下文引擎个性化推荐

## 🛠️ 部署配置

### Vercel部署
```json
{
  "rewrites": [
    { "source": "/api/(.*)", "destination": "/api/$1" },
    { "source": "/(.*)", "destination": "/index.html" }
  ]
}
```

### Netlify部署
```toml
[build]
  command = "pnpm build"
  publish = "dist"

[[redirects]]
  from = "/api/*"
  to = "/.netlify/functions/:splat"
  status = 200

[[redirects]]
  from = "/*"
  to = "/index.html"
  status = 200
```
