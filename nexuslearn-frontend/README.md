# Web3.0 AI学习平台

## 🌟 项目简介

这是一个集成了Web3.0、AI和现代化设计的全栈学习平台，提供个性化的学习体验、智能社区互动和区块链技术集成。

## ✨ 核心功能

### 🧠 知识仓库系统
- **学习资料管理**: 集中管理各类学习资源
- **代码模块库**: 可复用的代码片段和组件
- **上下文引擎**: 基于用户偏好的智能推荐
- **高级搜索引擎**: AI驱动的智能搜索功能

### 🤝 AI社区系统
- **朋友圈**: 学习动态分享和互动
- **兴趣圈**: 基于共同兴趣的社群交流
- **话题圈**: 专题讨论和知识分享
- **AI助手**: 智能问答和学习建议

### 📚 学习管理中心
- **AI驱动学习路径**: 个性化学习计划生成
- **自适应评估**: 实时难度调整和进度跟踪
- **智能练习系统**: 题目推荐和实时反馈
- **进度可视化**: 学习轨迹图表展示

### ⛓️ Web3.0集成
- **钱包连接**: MetaMask、WalletConnect支持
- **NFT学习证书**: 区块链认证的学习成就
- **智能合约**: 去中心化应用功能
- **代币经济**: 学习激励和治理机制

## 🎨 设计特色

### 现代化UI
- **玻璃拟态效果**: 半透明背景配合模糊效果
- **渐变色彩系统**: 多层次的渐变色彩搭配
- **动态粒子背景**: 增强视觉体验的动画效果
- **响应式布局**: 完美适配移动端和桌面端

### 交互体验
- **平滑动画**: 所有交互都有流畅的过渡效果
- **实时反馈**: 用户操作立即得到视觉反馈
- **智能提示**: AI驱动的个性化建议和提醒
- **无障碍设计**: 支持键盘导航和屏幕阅读器

## 🚀 快速开始

### 环境要求
- Node.js >= 16.0.0
- npm 或 pnpm
- MetaMask浏览器插件（用于Web3.0功能）

### 安装依赖
```bash
# 使用 pnpm（推荐）
pnpm install

# 或使用 npm
npm install
```

### 环境配置
```bash
# 复制环境变量模板
cp .env.example .env

# 编辑配置文件，添加必要的API密钥
vim .env
```

### 开发环境启动
```bash
# 启动开发服务器
pnpm dev

# 或使用 npm
npm run dev
```

访问 http://localhost:5173 查看应用

### 生产环境构建
```bash
# 构建项目
pnpm build

# 预览构建结果
pnpm preview
```

## 🌐 部署指南

### Vercel部署
```bash
# 安装Vercel CLI
npm i -g vercel

# 部署到Vercel
vercel --prod
```

### Netlify部署
```bash
# 安装Netlify CLI
npm i -g netlify-cli

# 部署到Netlify
netlify deploy --prod
```

## 📱 功能测试

### 1. 钱包连接测试
1. 确保已安装MetaMask插件
2. 点击右上角"连接钱包"按钮
3. 选择MetaMask或其他钱包
4. 授权连接请求

### 2. 学习路径测试
1. 进入"学习管理"页面
2. 设置学习目标
3. 添加核心技能标签
4. 生成AI学习计划

### 3. 智能练习测试
1. 配置练习参数
2. 选择练习主题和难度
3. 开始智能练习
4. 查看AI反馈和建议

### 4. 社区功能测试
1. 发布学习动态
2. 参与话题讨论
3. 使用AI助手
4. 加入兴趣圈子

## 🛠️ 技术栈

### 前端技术
- **Vue 3**: 渐进式JavaScript框架
- **TypeScript**: 类型安全的JavaScript超集
- **Vite**: 快速的构建工具
- **Element Plus**: Vue 3组件库
- **Tailwind CSS**: 实用优先的CSS框架

### 区块链集成
- **Web3.js**: Ethereum JavaScript API
- **Ethers.js**: Ethereum库和工具
- **WalletConnect**: 去中心化钱包连接
- **IPFS**: 分布式文件存储

### AI服务
- **OpenAI API**: GPT模型集成
- **自定义AI服务**: 个性化推荐和评估
- **机器学习算法**: 自适应学习路径

### 开发工具
- **ESLint**: 代码质量检查
- **Prettier**: 代码格式化
- **Vitest**: 单元测试框架
- **Vue DevTools**: Vue开发调试工具
