# Web3.0 AI学习平台 - 一键启动器使用指南

## 🚀 快速启动

### Windows系统
1. **双击运行**: `启动Web3学习平台.bat`
2. **Python启动**: `python start_web3_learning.py`

### macOS/Linux系统
1. **终端运行**: `bash start_web3_learning.sh`
2. **Python启动**: `python3 start_web3_learning.py`
3. **完整版启动**: `python3 launch_platform.py`

## 🛠️ 故障排除

### 常见问题

#### 1. 依赖安装失败
```bash
# 清理缓存后重试
npm cache clean --force
npm install
```

#### 2. 端口被占用
```bash
# 修改端口
npm run dev -- --port 3000
```

#### 3. 浏览器无法打开
- 手动访问: http://localhost:5173
- 检查防火墙设置
- 验证网络连接

#### 4. Web3功能异常
- 安装MetaMask插件
- 连接到正确的网络
- 检查钱包授权
